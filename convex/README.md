# Convex Functions

This directory contains Convex backend functions.

## Setup Required

To complete the Convex setup, you need to:

1. Run `npx convex dev` in a terminal
2. Login with GitHub when prompted
3. Select or create a project named 'task-manager'
4. The deployment URL will be automatically added to your `.env.local` file

The `_generated/` folder will be created automatically when you run `npx convex dev`.
